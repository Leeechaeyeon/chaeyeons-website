웹표준을 준수하여 만든 사이트입니다.

이 사이트를 만든 이유

이 웹 사이트는 웹 표준을 준수하여 만든 사이트로써, 페이지 내의 이미지는 직접 찍은 이미지를 이용했습니다.
본 사이트 내의 이미지는 무단 복제나 사용을 금합니다.

웹 표준

마우스오버 시 상세 내용이 나타납니다.

웹 표준(Web Standard)이란 요약해서 '웹에서 사용되는 표준 기술이나 규칙'을 의미합니다.

웹 접근성

마우스오버 시 상세 내용이 나타납니다.

웹 취약계층(장애인, 고령자 등)및 모든 사용자가 어떠한 기술이나 환경에서도 전문적인 능력 없이 웹사이트에서 제공하는 모든 정보를 동등하게 접근하고 이해할 수 있는 권리입니다.

텍스트 유형 01

웹 표준[Web Standards]이란?

여러 종류, 버전이 있는 브라우저 간에 상호 호환성을 위해 W3C가 권고한 표준적인 기술이나 규칙을 말합니다.
즉, 사용자가 어떤 브라우저나 기기를 사용하더라도 홈페이지 화면을 동일하게 볼 수 있도록 표준화된 방식으로 작업하는 것을 의미합니다.

카드 유형 01

웹 표준을 준수하는 방법

웹 표준 관련 기술에는 3가지가 있습니다.
구조 (X)HTML, 표현 CSS, 동작 Script 입니다.

01 DOCTYPE 선언

웹페이지의 최상단에 DOCTYPE을 선언하여 웹 페이지가 어떤 버전의 HTML, XHTML을 사용하는지 명시합니다.

02 시맨틱 태그 사용

시맨틱 태그는 HTML5에서 새로 추가된 태그로, <header>, <nav>, <section>, <footer> 등의 태그를 사용하여 웹 페이지의 구조를 더욱 명확하게 표현할 수 있다.

03 CSS 스타일시트 사용

CSS 스타일시트를 사용하여 디자인과 레이아웃을 분리하고, 웹 페이지의 내용과 디자인을 분리하여 유지보수 및 확장성을 높일 수 있다.

04 웹 접근성 고려

웹 페이지를 만들 때 모든 사용자가 쉽게 접근할 수 있도록 웹 접근성을 고려해야 합니다. 웹 접근성은 신체적, 환경적 조건에 관계없이 인터넷을 통해 정보에 접근하고 이용할 수 있도록 하는 것을 말합니다.

이미지 텍스트 유형 01

웹 표준의 장점

웹 페이지를 작성할 때 웹 표준을 준수하면 다음과 같은 효과를 누릴 수 있게 된다.

  • 웹 페이지의 호환성
  • 효율적인 마크업
  • 검색 엔진 최적화 (SEO)
  • 유지보수 용이 및 비용 절감 효과
  • 웹 접근성 향상
